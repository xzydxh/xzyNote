    # 内置实体特殊处理组件

    • 使用效果：
    • 代码：
     <base-dialog
      v-if="roleDialog"
      title="复制权限"
      width="800px"
      :visible.sync="roleDialog"
    >
      <base-form
        ref="roleForm"
        :data="roleForm"
        :rulesOuter="roleRules"
        class="role-form"
      >
        <el-row :gutter="24">
          <el-col :span="12">
            <el-form-item
              prop="roleName"
              :label="
                entity.systemIdentification == 36 ? '职务名称' : '岗位名称'
              "
            >
              <el-input disabled v-trim v-model="roleForm.roleName"></el-input>
            </el-form-item>
          </el-col>
          <el-col :span="12">
            <el-form-item prop="description" label="描述">
              <el-input
                v-trim
                disabled
                v-model="roleForm.description"
              ></el-input>
            </el-form-item>
          </el-col>
        </el-row>
        <el-row :gutter="24">
          <el-col :span="12">
            <el-form-item
              prop="copyRoleId"
              :label="
                entity.systemIdentification == 36 ? '职务标识' : '岗位标识'
              "
            >
              <base-select
                v-model="roleForm.copyRoleId"
                filterable
                clearable
                placeholder="请选择"
              >
                <el-option
                  v-for="item in rolesOptions"
                  :key="item.id"
                  :label="item.object_name"
                  :value="item.id"
                >
                </el-option>
              </base-select>
            </el-form-item>
          </el-col>
        </el-row>
      </base-form>
      <div slot="footer" class="dialog-footer">
        <base-button
          class="major"
          @click="submitForm()"
          :loading="submitLoading"
          >确 定
        </base-button>
        <base-button class="ordinary" @click="dialogCancel()"
          >取 消
        </base-button>
      </div>
    </base-dialog>
    <el-dialog title="选择作业指南" :visible.sync="homeworkDialog">
      <entityObjectselect ref="homeworkEntity"> </entityObjectselect>

    • 说明：

      实例管理特殊的处理逻辑，如弹框。

    • 参数及方法：

      props：

      参数 说明 类型 可选值 默认值
      entity 当前字段绑定的值 string —— ——
      displayId 视图id object —— ——

      events：

      事件名称 说明 回调函数
      rightHandleClick 获取复制权限下拉框的数据 实体为岗位或职务触发
      permissionDialog 打开分配权限弹框 右键点击复制权限触发
      blurFn 复制权限弹框确定 复制权限弹框点击确定触发
      handleClick 复制权限弹框取消 复制权限弹框取消触发
      handleChange 打开选择作业指南弹框 点击选择作业指南条目按钮触发
