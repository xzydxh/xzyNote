    实例回收站按钮组件

    # 实例回收站按钮组件

    # /components/aos-public/aos-button/recycleBtn.vue

    # 实例回收站

    • 使用效果：

    • 代码：
    <base-button
    :disabled="ids.length == 0"
    @click="reductionObj()"
    >还原
  </base-button>
  <base-button
    v-if="NormalBtn.length"
    @click="fullDelete()"
    :disabled="ids.length == 0"
    >彻底删除
  </base-button>
  <base-button @click="backPage()">返回</base-button>
    • 说明：

    从目前的开发来看，实例回收站列表上方的按钮一直都是固定的。所以，该处的按钮单独封装成了组件。

    • 使用范例（出处：src\components\aos-public\aos-table\aos-business-table.vue)：
    <recycle-btn 
	:entity="entity" 
    :ids="ids" 
    :btnsArr="buttons" 
	v-show="isDelete == 1 && defaultBtn" 
	@remove-refresh="removeRefresh" 
/>
<base-button v-show="isDelete != 1 && defaultBtn" class="right_btn" @click="toRecycleBin()">回收站 </base-button>

    Props

    参数 说明 类型 可选值 默认值
    isDelete 该值等于1时，代表页面实例页面是处在回收站列表的 number —— ——
    entity 实体对象 object —— ——
    ids 列表数据项被选中的id集合 array —— ——
    btnsArr 后端返回的按钮数据数组 array —— ——

    Events

    事件名称 说明 回调函数
    remove-refresh 回收站页面中，“还原”、“彻底删除”、“返回”操作触发 “还原”、“彻底删除”、“返回”操作的回调函数
