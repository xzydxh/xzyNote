    # /components/aos-public/aos-module/aos-deptselect.vue

    # 行编辑部门组件

    • 使用效果：
    • 代码：
    <div class="box">
  <el-input
  ref="tem"
  :class="{'focus':iSfocus}"
  v-model="name"
  :title="name"
  readonly
  @click.native="handleClick"
  @dblclick.native="dblclick"
  @contextmenu.native="blurFn"
  @blur="blurFn"
  ></el-input>
  <BaseDeptSelect
      ref="dialogSelect"
      :options="rowEditOptions.options"
      @drawer-change="handleChange"
      @close="close"
  />
</div>

    • 说明：

      实例管理行编辑时，部门字段统一使用aos-deptselect显示和处理

    • 参数及方法：

      props：

      参数 说明 类型 可选值 默认值
      value 当前字段绑定的值 string —— ——
      rowEditOptions 字段的行编辑配置信息 object —— ——
      row 当前行数据 object —— ——
      cellData cellData object —— ——
      propCode 部门字段用于回显的数组ID string —— ——

      events：

      事件名称 说明 回调函数
      dblclick 双击输入框，打开选择部门弹框 双击输入框触发
      handleClick 点击表格获取焦点 单击表格触发
      blurFn 失去焦点 输入框失去焦点或右键触发
      handleChange 部门弹框点击确定触发 部门弹框点击确定触发

    # BaseDeptSelect子组件

    • 说明：引用base-deptselect.vue文件

      --props：

      options：字段配置信息

      --events：

      show：接收一个数据，用于反显选中数据

      handleSubmit：弹框点击确定将选中项名称和数组ID返回给父组件

    ​ props：

    参数 说明 类型 可选值 默认值
    cellData 当前列的信息 object —— ——
    index 下拉复制行的索引 number —— ——

    events：

    事件名称 说明 回调函数
