    # Attributes（参数）

    参数 说明 类型 可选值 默认值
    title 弹窗标题 string — —
    columnDefs 表头信息 array — —
    layout 渲染的组件（左树，中树，右表） string lTree,mTree,list（左树，中树，右表） list（默认只渲染列表）
    initFn 初始化运行的方法（左树，中树，右表） string lTree,mTree,list（左树，中树，右表） lTree,mTree,list（默认全部运行）
    icon 自定义多选+号 boolean — false
    singleSelection 单选，多选 string "1"单选 大于"1"多选 默认不显示可选框
    maxSelectable 最多可选几个 string — —
    dataName 需要哪个参数的数据就传哪个(:dataName="['id']") array — —

    # Events（方法）

    参数 说明 参数
    custom 接收自己定义的想要的参数数组对象 （对象）
    change 多选框选中的数据 selection

    # Methods （初始化方法）

    参数 说明 参数
    init 调用弹窗事件 使用：init(listDate,columnDate,middleTree,leftTree){//列表数据、表头数据、中树、左树 对象（1.listDate.way;列表方法 2.listDate.param；列表条件）
    initEcho 选中的数据回显 this.$refs.aosDialog.initEcho(对象)
